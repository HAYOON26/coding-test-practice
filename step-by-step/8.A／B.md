
- 두 정수 A와 B를 입력받은 다음, A/B를 출력하는 프로그램을 작성하시오.  

## 현식의 코드

```java
import java.util.Scanner;

public class Main {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        Double A = sc.nextDouble();
        Double B = sc.nextDouble();

        System.out.println(A / B);
    }

}
```
